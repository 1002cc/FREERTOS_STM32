
# 智能控制


* 基于STM32实现的智能控制项目，可通过微信小程序和蓝牙助手连接控制,也可以通过实体按键控制


## 1.硬件介绍
```
    STM32开发板f407芯片
    传感器模块:
            ESP8266WIFI模块
            蓝牙2.0模块
            超声波模块HC_05
            矩阵按键
            烟雾报警器MQ2
            MCP6050电子陀螺仪
            SG90舵机
            OLED屏0.96寸
            ......
```
## 2.技术介绍
* FREERTOS嵌入式实时操作系统
* STM32标准库
* C语言
